[AskAGenie](https://askagenie.com/) is an AI-driven wrapper that bridges the gap between conversational intent to help with business ideas and the resources available to start them. Unlike standard chatbots, this platform utilizes a sophisticated Go-backend to parse user prompts, identify specific needs, and query relevant product databases to provide instant, curated shopping suggestions.

![ask-a-genie](/src/assets/ask-a-genie-screenshot.png)

- **Built to Last**: Built with a basic HTML, CSS, and Javascript frontend for a seamless user experience that doesn't expire with a new framework updateand a robust Go backend hosted on AWS.

- **Custom Built Backend**: Developed a custom parsing logic that extracts intent from OpenAI’s API responses, allowing the system to bridge the "black box" of AI with real-world inventory.

- **Value Added**: By converting vague queries into specific product links, the site reduces the friction of the "search-to-purchase" pipeline, offering a blueprint for the future of personalized e-commerce.
